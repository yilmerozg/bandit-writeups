# Level 1

## Görev
`-` adlı dosyanın içindeki şifreyi bul.

## Çözüm
```bash
ls        # dosyayı gör: -
cat ./-   # ./ ekleyerek dosyayı oku
```

## Öğrendiklerim
- `-` adlı dosyalar özeldir, terminal bunu parametre olarak algılar
- `./` ekleyerek "bu klasördeki dosya" olduğunu belirtebiliriz
- Bu tür istisnai dosya adları gerçek sistemlerde de karşılaşılabilir

## Sonuç
Şifre başarıyla bulundu ve Level 2'ye geçildi.
