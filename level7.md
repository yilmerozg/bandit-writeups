# Level 7

## Görev
`data.txt` dosyasında "millionth" kelimesinin yanındaki şifreyi bul.

## Çözüm
```bash
grep millionth data.txt
```

## Öğrendiklerim
- `grep` komutu dosya içinde kelime arar
- Büyük dosyalarda tek tek okumak yerine grep kullanılır
- SOC çalışmasında log dosyalarında IP, kullanıcı adı ararken çok kullanılır

## Sonuç
Şifre başarıyla bulundu ve Level 8'e geçildi.
