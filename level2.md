# Level 2

## Görev
`--spaces in this filename` adlı dosyanın içindeki şifreyi bul.

## Çözüm
```bash
ls                              # dosya adını gör
cat "./--spaces in this filename"  # tırnak ve ./ ile oku
```

## Öğrendiklerim
- Dosya adında boşluk varsa tırnak içine alınmalı
- Dosya adı - ile başlıyorsa ./ eklenmeli
- Tab tuşu dosya adını otomatik tamamlar

## Sonuç
Şifre başarıyla bulundu ve Level 3'e geçildi.
